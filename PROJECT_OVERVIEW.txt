└──────────────────────────────────────────────────────────────────────────────┘

1. SETUP (First time only)
   > npm run setup

2. CONFIGURE
   > cp .env.example .env
   > Edit .env and set BASE_URL

3. VALIDATE FLOW
   > npm run validate flows/demo-navegacion.json

4. GENERATE VIDEO
   > npm run generate -- flows/demo-navegacion.json

5. VIEW OUTPUT
   > output/demo-navegacion/demo-navegacion.mp4


┌──────────────────────────────────────────────────────────────────────────────┐

└──────────────────────────────────────────────────────────────────────────────┘

npm run setup              Install all dependencies + Chromium
npm run generate           Generate video from flow
npm run validate           Validate flow JSON file
npm run remotion:preview   Open Remotion Studio preview
npm run setup:browsers     Download Chromium browser
npm run dev                Alias for remotion:preview

Generate options:
  --headed               Show browser during capture
  --headless             Run browser in background (default)
  --skip-render          Only capture, don't render video
  --base-url <url>       Override base URL
  --output <name>        Custom output filename
